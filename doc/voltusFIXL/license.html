Voltus-Fi Licensing Information

You need the following licenses to use the Voltus-Fi Custom Power Integrity Solution software:

License License Number Description
Virtuoso Framework License (running Voltus-Fi-XL in the GUI mode requires checkout availability of the following license sets)

Cadence Design Framework II (DFII)

111

This license must be checked out at Virtuoso startup.

Virtuoso Analog Design Environment XL (ADE XL)

95210

This license is required to run Voltus-Fi-XL from the Virtuoso Analog Design Environment (ADE L/ADE XL).
Simulator License (you need one of the following licenses for performing simulation)

Spectre Accelerated Parallel Simulator (Spectre APS) with Spectre Power

91400

This license must be purchased separately.

The Spectre Power option enables EMIR analysis with the Spectre APS base product.

Spectre eXtensive Partitioning Simulator (XPS) with Spectre Power

91400

This license must be purchased separately.

The Spectre Power option enables EMIR analysis with the Spectre XPS base product.
Voltus-Fi License (the Voltus-Fi Custom Power Integrity Solution software includes the following products. When you run a command to use a product, a license is checked out)

Voltus-Fi Custom Power Integrity Solution-L (VTS-Fi-L)

VTS400

License string: Voltus_Power_Integrity_Fi_L

Uses the extracted view to perform IR drop and Electromigration (EM) analysis. It uses the results of the simulation performed using the vavo-db or the psf data.

Voltus-Fi Custom Power Integrity Solution-XL (VTS-Fi-X)

VTS500

License string: Voltus_Power_Integrity_Fi_XL

Performs multi-mode simulation (MMSIM) electromigration and voltage drop (EMIR) analysis for power and signal nets using the simulation database generated by Spectre APS or Spectre XPS. It is tightly integrated within Virtuoso.

With this license, you can also run Voltus-Fi Custom Power Integrity Solution-L (VTS-Fi-L).

Voltus-Fi Custom Power Integrity Solution-AA (VTS-Fi-AA)

VTS501

License string: Voltus_Power_Integrity_Fi_AA

Supports the following advanced features:

  • Hierarchical EMIR Analysis
  • Hierarchical SHE Analysis
Beta License

VPS_NG100

VPS_NG100

The following features are beta features and available only with this license:

  • EM analysis on the N3 process node
  • ESD analysis
