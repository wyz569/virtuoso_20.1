Getting Started with the Track Pattern Assistant

The Track Pattern assistant acts as a guidance method to capture design rules and design intent for creating wires and it is displayed as tracks on the canvas. It lets you choose the width spacing patterns that apply across the entire cellview, referred to as global area, or in a specific local area.

You can use this assistant to enable the predefined WSSPDefs for the global grid and choose the active pattern and optional wire type for enabled WSSPDefs from the allowed patterns. By default global regions are created using the Track Pattern assistant. You can also create local regions using the Track Pattern assistant.
