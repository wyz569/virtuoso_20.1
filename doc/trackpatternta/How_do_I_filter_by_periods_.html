Product Version ICADVM20.1, October 2020

How do I filter by periods?

If the Show All check box is selected, the Periods filter list box displays the periods for the enabled WSSPDefs in the Track Pattern table. Otherwise, only the periods for the enabled WSSPDefs are displayed.

To show only WSSPDefs with a specific period:

  • Click the period in the Periods list box.

The period is selected. Only WSSPDefs with a period matching a selected period are displayed in the Track Pattern table. Multiple periods can be selected.

To deselect a period:

  • Click the period.

The filter is removed for the period.
