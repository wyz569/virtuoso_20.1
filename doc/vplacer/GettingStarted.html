Getting Started with Virtuoso Placer

Virtuoso® Placer is a unified placement solution that can be used to achieve quick, high-quality, DRC-correct placement of standard cells and devices. It supports placement of different types of designs, such as digital designs with only standard cells, analog designs with only devices, and mixed designs that contain a mixture of both. It can also be used to generate connectivity and constraint-driven device placements.

Placer combines several advanced features such as snap patterns, width spacing patterns, and multi-patterning into a single automated flow. At the core of the Placer is the row template infrastructure that lets you create rows, which, in turn, define how standard cells and devices are placed. Rows, if made compatible with snap patterns, help achieve correct-by-construction placement.

Placer also brings together under its umbrella several existing and new tools that utilize the row template infrastructure. For example, you can use Virtuoso® Custom Digital Placer to place custom digital designs and Modgens to construct blocks of analog devices. The new automatic and assisted placement and post-processing tools can be used to fill the design with dummy devices to meet density DRC rules and to achieve better matching.

Placer is available in the Virtuoso Layout EXL cockpit in advanced node releases.

Before using the placer, ensure that the circuit components and connectivity are generated by using the Virtuoso Layout Suite XL layout editor. The placer uses the schematic design as the connectivity source for placing the components.

Virtuoso Placer Flow

Virtuoso Placer is a unified, powerful placement solution. Its key benefits are:

  • Accelerated layout productivity for advanced node placement.
  • Unified GUIs and use-models between the various placers.
  • Improved usability of the set up and placement commands.

Virtuoso Placer Commands

The following placement commands are available under the Place menu:

The following table provides a brief description of the commands in the Place menu, their corresponding icons in the Placement toolbar, and a link to related documentation.

Icon

Command Name

Description

Pin Placement

Opens the Pin Planner, which lets you assign and refine pin constraints and their attributes, and the Pin Optimizer, which positions pins in a manner that helps obtain the shortest possible net length at a particular level in the design. Use this command to preplace pins, independent of the main placement run.

For more information, see Pin Planning.

Update Placement Status

Lets you update the placement status of instances or pins that are loaded in Layout XL for the first time, or which have been edited outside the Layout XL or higher tiers.

Row Template Manager

Lets you define row templates, which can later be used to generate rows in the layout canvas.

For more information, see Creating Rows.

Placement Planning

Lets you plan the placement region, generate rows in which the components will be placed, and select additional substrates to be inserted during placement.

For more information, see Placement Planning.

Boundary Cells Placement

Places boundary cells around core cells to protect the core cells from the undesired effects that are caused when adjacent cells are placed within close proximity.

For more information, see Placing Boundary Cells.

Placement Options

Provides options to customize assisted and automatic placement. This form can also be invoked by clicking More Options in the Automatic Placement form. You can make the required placement settings and then run the Virtuoso Placer.

For more information, see Placement Options.

Swap Tap Cells

Detects tap cells with maximum spacing violations and replaces them with tap cells that have valid maximum spacing values.

For more information, see Swapping Tap Cells.

Auto Place

Lets you place devices, standard cells, or a mix of both in rows defined in the layout canvas.

For more information, see Automatic Placement.

Resolve Overlaps

Resolves device overlaps in rows.

For more information, see Performing Assisted Placement.

Abut Instances

Abuts selected or all instances in the design.

For more information, see Performing Assisted Placement.

UnAbut Instances

Unabuts selected or all instances in the design.

For more information, see Performing Assisted Placement.

Swap Instances

Swaps the positions of selected devices.

For more information, see Performing Assisted Placement.

Swap Rows

Swaps the positions of the selected rows.

For more information, see Performing Assisted Placement.

Snap to Grids/Rows

Snaps the selected devices to appropriate grids or rows.

For more information, see Performing Assisted Placement.

Adjust PR Boundary

Adjusts the boundary by deleting any extra rows in which devices are not placed.

For more information, see Performing Assisted Placement.

Modgen

Opens a submenu that provides commands to create and edit Modgen constraints.

For more information, see Module Generator User Guide.

Filler

Opens a submenu that provides options to:

  • Insert filler cells - For digital designs
  • Insert dummy fill - For analog designs

For more information, see Insert/Delete Dummy Fill.
