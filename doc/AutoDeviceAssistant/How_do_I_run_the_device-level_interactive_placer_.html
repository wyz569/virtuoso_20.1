Auto Device Placement and Routing Quick Help
Product Version ICADVM20.1, October 2020

How do I run the device-level interactive placer?

To run the device-level interactive placer:

  1. Open the Placer tab of the Auto Device P&R assistant.
  2. Select Context Menu to display a shortcut menu that lets you perform context-sensitive operations on devices and device groups directly in the layout canvas. For more information about these operations, see:
    • How do I group devices?
    • How do I specify whether devices must be stacked vertically or spread horizontally?
    • How can I control the direction in which devices are placed when they are moved?
  3. Select Row Snapping to display row snapping-related details in the Information pane during interactive device placement.
  4. Select WSP Snapping to display SP and WSP snapping-related details in the Information pane during interactive device placement.

    Related Content: Virtuoso Automated Device Placement and Routing Flow Guide
