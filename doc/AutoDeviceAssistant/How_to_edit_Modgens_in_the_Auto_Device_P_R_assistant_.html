How to edit Modgens in the Auto Device P&R assistant?

To edit a Modgen:

  1. Open the Constraints tab of the Auto Device P&R assistant.
  2. Expand the required category in the View and Edit Constraints section. 
  3. Right-click the required Modgen constraint.
  4. Choose Edit Group. The Auto Device Array form is displayed.
  5. Specify the required changes in the Placement and GuardRing tabs. 
  6. Click Apply.
