DRD Interactive Compactor

This section describes the DRD interactive compactor, which you can use to compact the selected objects to the closest violation-free location according to the spacing rules defined in the Virtuoso technology file, or a custom spacing, which you can specify in the DRD Compactor Options form. You can also compact the selected objects against a specified target object (an anchor).

The compactor is available only in VLS XL and VLS GXL tiers.
For a quick overview of DRD Compactor, see Using DRD Compactor.

This chapter contains the following topics:

  • Overview of DRD Compactor Features
  • Prerequisites for Using the Compactor

Overview of DRD Compactor Features

The compactor supports the following features:

  • Compaction of vias and orthogonal and 45-degree rectangles, polygons, and wires
    For examples of compaction of 45-degree shapes, see
    • Example 8: Compacting 45-degree wires in Virtuoso Partial Select mode
    • Example 9: Compacting 45-degree polygons in Virtuoso Partial Select mode.
  • Compaction of instances
    The compactor does not compact objects inside an instance against each other.
  • 1-D orthogonal compaction in the left, right, top, or bottom directions using icons in the DRD Compactor toolbar or the default bindkeys
  • Compaction with move and stretch effect, which
    • Moves shapes if Virtuoso Full Select mode is enabled as shown in Example 5: Compacting objects in Virtuoso Full Select mode
    • Stretches shapes if Virtuoso Partial Select mode is enabled as shown in Example 6: Compacting objects in Virtuoso Partial Select mode
  • Compaction according to the single- and multi-layer spacing constraints supported by DRD Enforce
  • Placement of shapes next to each other after compaction if no spacing constraints are defined in the technology file and no custom spacing is specified
  • Undo compaction results using the Undo command in Virtuoso
  • Compaction in several modes as described in Compaction Mode
  • Setting the compactor options in the The DRD Compactor Options Form
  • Running the compactor in two ways:
    • Using the Compactor GUI
    • Using SKILL Functions and Bindkeys
Compactor does not support path shapes and derived layers.

Prerequisites for Using the Compactor

Before using the compactor, do the following:

  1. Check that your technology file includes the required spacing constraints for the layers you want to use for compaction.
  2. If you are compacting hierarchical objects, such as instances:
    1. In the DRD Options form, choose Hierarchy Range as current to bottom.
    2. Set the display hierarchy depth to a value greater than zero using the display stop level options.
      You set the hierarchy depth using one of the following display stop level options in Virtuoso:
      • The icon in the Options toolbar.
      • The Display Options form, which is launched by choosing Options — Display.

The DRD Compactor Options Form

You can choose the compaction mode, the spacing option, and the anchor mode using the DRD Compactor Options form.

You can change the default compaction mode using the drdEditCompactorMode environment variable.

Spacing Options

Objects are compacted or spread according to the spacing option chosen in this group box.

  • DRD Enforce spacing (default)
    Compacts and spreads according to the spacing constraints that are defined in the technology file and supported by DRD Enforce.
  • User spacing
    Compacts and spreads according to a custom spacing value. Select the field and type the required spacing value.
    For an application of this spacing option, see Example 1: Compacting objects with different spacing values so that all the objects are spaced equally.
  • Apply to Active Layer in Palette applies User spacing to only the shapes on the currently active layer.
    You can change the default spacing option using the drdEditCompactorSpacingMode environment variable. You can set the default custom spacing value using the drdEditCompactorUserSpacing environment variable. You can set the spacing to apply to active layer shapes only using the drdEditCompactorActiveLayerOnly environment variable.

Anchor Mode

Anchors are the target objects in the layout against which other objects are compacted.

You can set the default anchor mode using the drdEditCompactorAnchorMode environment variable.
  • Use last shape in direction of compaction (default)
    Compaction occurs only within a set of selected objects. The last object in the direction of compaction is used as the target for compaction. For example, if you are compacting left, the leftmost object is used as the target. The target object remains static; the remaining objects are compacted or spread against the target object.
    For an application of this mode, see Example 2: Compacting within a selected set of objects.
  • Use first unselected shape in direction of compaction
    The first unselected object found in the direction of compaction is used as the target. The selected objects are compacted or spread against the target object.
    If there is no unselected object in the direction of compaction within the visible area of the layout canvas, the selected objects are compacted against the edge of the layout window.
    For an application of this mode, see Example 3: Compacting the selected objects against an unselected object.
  • Use DRD Targets
    The selected objects are compacted against a DRD Target object. The target object can be inside or outside the visible area of the layout canvas.
    If your design does not contain DRD targets, you must set an object as a target before you proceed with the compaction. For more information, see Setting DRD Targets.
  • Maintain relative positions of selected objects
    Maintains the original spacing between the selected objects during compaction. For example, if you have selected three objects for compaction, the object closest to the anchor is compacted against the anchor and the original spacing between the three objects remains unchanged.
    For an application of this field, see Example 4: Maintaining the original spacing between the selected objects during compaction.
    This field is enabled for selection only when one of the following anchor modes is selected:
    • Use first unselected shape in direction of compaction
    • Use DRD Targets
      You can select this field by default using the drdEditCompactorInterOnly environment variable.

The DRD Compactor Toolbar and Bindkeys

If the DRD Compactor toolbar is not visible in the Virtuoso toolbars section, choose Window — Toolbars — DRD Compactor to make it visible.

You can click the icons on the toolbar to do the following:

  • Launch the DRD Compactor Options form.
    For information about launching the form, see Launching the DRD Compactor Options Form.
  • Compact objects in different directions.
    As an alternative, you can also use the default compactor bindkeys.

The following figure shows the functionality of the icons, the default compactor bindkeys, and the icon-to-bindkey mapping.

You can also customize the bindkeys using the hiSetBindKey command, as shown in the following example.

hiSetBindKey("Layout", "<Ctrl>Left", "drdCompactSelSet(\"L\", \"compactAndSpread\")" )
hiSetBindKey("Layout", "<Ctrl>Right", "drdCompactSelSet(\"R\", \"compactAndSpread\")" )
hiSetBindKey("Layout", "<Ctrl>Up", "drdCompactSelSet(\"T\", \"compactAndSpread\")" )
hiSetBindKey("Layout", "<Ctrl>Down", "drdCompactSelSet(\"B\", \"compactAndSpread\")" )

In the example above, the compact and spread mode has been assigned to all the bindkeys. Each bindkey calls the drdCompactSelSet SKILL function. Ctrl+left arrow compacts left, Ctrl+right arrow compacts right, Ctrl+top arrow compacts up, and Ctrl+bottom arrow compacts down.

Running the Compactor

This section provides the generic steps to run the compactor. For different examples of compaction and the steps specific to each example, see Examples of Compaction.

You can run the compactor in two ways:

  • Using the Compactor GUI
  • Using SKILL Functions and Bindkeys
    Before running the compactor, ensure that the prerequisites for using the compactor are met. For more information, see Prerequisites for Using the Compactor.

Using the Compactor GUI

To run the compactor using the GUI, do the following:

  1. On the layout canvas, select the objects that you want to compact.
  2. Choose Options – DRD Compactor from the layout window menu bar.
  3. In the DRD Compactor Options form, do the following:
    1. Select the Compaction Mode.
    2. Select the Spacing Options.
    3. Select the Anchor Mode.
    4. Click OK after setting the compactor options as required.
  4. On the DRD Compactor toolbar, click the appropriate icon to compact the selected objects in the required direction.

Specifying the Number of Shapes for Compaction

You can use the maxShapesAllowedForCompaction environment variable to set the number of shapes that can be compacted. The default value is 500.

If the number of shapes involved in compaction is greater than the value set in this environment variable, the compactor does not compact the selected objects and displays a warning message in the CIW.

Interrupting a Compaction Job

If a compaction job takes too long to complete, you can manually interrupt it by pressing Ctrl+C. A message stating that compaction was stopped by the user is displayed in CIW.

Using SKILL Functions and Bindkeys

To run the compactor using SKILL functions and bindkeys, do the following:

  1. On the layout canvas, select the objects that you want to compact.
  2. To compact the objects, run the drdCompactSelSet SKILL function using one of the following alternatives:
    • Type the SKILL function in the CIW and specify the direction of compaction using the s_direction parameter. For example, drdCompactSelSet('top) compacts the selected objects in the upward direction.
      Other parameters for the SKILL function are optional. If you do not set the optional parameters, the compactor uses the default compaction mode, that is, compact and spread.
    • Assign the SKILL function to a bindkey by specifying a direction (for more information, see The DRD Compactor Toolbar and Bindkeys) and press the appropriate bindkey to run the SKILL function.
      You can compact objects in batch mode using the drdCompactObjects SKILL function.

Examples of Compaction

This section illustrates different examples of compaction. Each example contains the before-and-after compaction scenarios, along with the scenario-specific steps to run the compactor. For the generic steps to run the compactor, see Running the Compactor.

Before running the compactor, ensure that the prerequisites for using the compactor are met. For more information, see Prerequisites for Using the Compactor. You must launch the DRD Compactor Options form and change the settings in the form, as described in each example. For more information about how to launch the form, see Launching the DRD Compactor Options Form. Click OK after setting the compactor options as required.

Example 1: Compacting objects with different spacing values so that all the objects are spaced equally

Before Compaction After Compaction

Rectangles A and B are spaced at 0.5 micron; B and C are spaced at 1.5 microns.

The spacing between A and B, and B and C is corrected to 0.8 micron.
  1. Select A, B, and C, as shown in the Before Compaction figure.
  2. In the DRD Compactor Options form:
    • Ensure that Compaction Mode is set to Compact and Spread.
    • In Spacing Options, select User spacing and type 0.8.
  3. On the DRD Compactor toolbar, click Compact selected objects left.
    As shown in the After Compaction figure, the compactor does the following:
    • Spreads rectangles A and B by 0.3 micron—from 0.5 to 0.8 micron
    • Compacts rectangles B and C by 0.7 micron—from 1.5 to 0.8 micron

Example 2: Compacting within a selected set of objects

Before Compaction After Compaction

Three of the selected rectangles need to be compacted against a selected rectangle.

Three selected rectangles are compacted against rectangle A.
  1. Select the four rectangles, as shown in the Before Compaction figure.
  2. In the DRD Compactor Options form:
    • Ensure that Compaction Mode is set to either Compact and Spread or Compact only.
    • In Anchor Mode, ensure that Use last shape in direction of compaction is selected.
  3. On the DRD Compactor toolbar, click Compact selected objects right.
    As shown in the After Compaction figure, the compactor moves three of the selected rectangles against rectangle A because it is the last selected shape in the direction of compaction.

Example 3: Compacting the selected objects against an unselected object

Before Compaction After Compaction

  • The selected rectangles need to be compacted against an unselected rectangle.

  • The selected rectangles, except B, are compacted against rectangle A.
  • B is compacted against the edge of the layout window.
  1. Select the five rectangles as shown in the Before Compaction figure.
  2. In the DRD Compactor Options form:
    • Ensure that Compaction Mode is set to either Compact and Spread or Compact only.
    • In Anchor Mode, select Use first unselected shape in direction of compaction.
  3. On the DRD Compactor toolbar, click Compact selected objects left.
    As shown in the After Compaction figure, the compactor moves the selected rectangles against the first unselected object found on the same layer, that is, rectangle A. Rectangle B is moved against the edge of the layout window because there is no unselected object in the direction of compaction to obstruct its movement.
