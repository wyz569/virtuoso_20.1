Interprocess Communication

hi interprocess communication functions have been replaced by ipc functions and are no longer available, except for hiSleep, which is an alias for ipcSleep.

For information about ipc functions, see Cadence Interprocess Communication SKILL Reference.

hiSleep

The hiSleep() function is an alias for the ipcSleep function.
