Licensing Utilities

This section describes the FlexNet and the Cadence utilities.

Note: For more information about the licensing utilities, see the Flexnet End Users Guide (flexnet_enduser.pdf). This guide is located at install_dir /share/license..

Function

Description

cdsIdent

Displays the version of libraries built into the executables

configure

Runs lic_config, mkclients, and rc.add

lic_config

Configures the license file

lic_error

Explains error message

lmdiag

Diagnoses license checkout problems.

lmdown

Shuts down the license daemons gracefully

lmgrd

The FlexNet license daemon

lmhostid

Returns the unique system-specific identifier used by the license manager to discern one computer from another

lmnewlog

Moves existing report log information to a new file name and starts a new report log file with existing file name.

lmremove

Releases any license that an abandoned or zombie process has locked and returns the license to the license pool of available licenses

lmreread

Forces the license daemons to reread the license file

lmstat

Returns information on the status of the license server and the licenses it serves. This is the most valuable utility.

lmswitch

Controls debug log location and size.

lmswitchr

Changes the report log file as specified in the options file (the enhanced log file, not the debug log file)

lmver

Displays the FlexNet version that a binary or library file uses

mkclients

Modifies the clients file

rc.add

Modifies the computer's startup file
