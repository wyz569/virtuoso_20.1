Heterogeneous Networks

Heterogeneous networks (more than one platform, such as a network with Sun and Windows NT workstations) do not really affect licensing. Because the license file is platform independent, you can use your licenses on any supported platform.

However, Cadence products are not platform independent and run on the platform specified by their DVDs. You can put the platform-independent directories and files in a shared location. The Cadence Installation Guide describes one method of installing multiple platforms on a file server.
