About This Reference

This reference is for Cadence system administrators--those providing the installation and licensing support for Cadence software on Linux platforms. Cadence system administrators must be familiar with UNIX® operating systems and a text editor.

This reference describes how to configure, monitor, and troubleshoot licensing. Here is a list of other documentation.
