Product Version ICADVM20.1, October 2020

What's New

New Setup Library Assistant

The new Setup Library assistant helps you define a project-specific (golden) setup for a block in ADE Assembler that you can reuse to create or modify the setup in Data View assistant. The Setup Library assistant can be accessed using ADE Assembler and ADE Verifier.
