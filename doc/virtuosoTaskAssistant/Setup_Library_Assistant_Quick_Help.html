Setup Library Assistant Quick Help

The Setup Library assistant provides a one-stop solution for:

  • Project Managers, who work on ADE Verifier for top-level verification, to define the project setup requirements and ensure full coverage of the specified requirements

  • Block Designers, who work on ADE Assembler for analog designing, to reuse the defined setup
