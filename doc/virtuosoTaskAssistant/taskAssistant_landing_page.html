Setup Library Task Assistant
Product Version ICADVM20.1, October 2020

2

taskAssistant_landing_page

The Setup Library assistant provides a one-stop solution for:

  • Project Managers, working on ADE Verifier for top-level verification, to define the project setup requirements and ensure full coverage of the specified requirements.
  • Block Designers, working on ADE Assembler for analog designing, to reuse the defined setup.
