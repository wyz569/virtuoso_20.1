Setup Library Task Assistant
Product Version ICADVM20.1, October 2020

How do I edit the existing setup?

To modify the specified setup (sweeps, corners, and verification spaces), ensure that the setup library cellview is enabled for editing.

If a lock symbol appears on Toggle Edit Mode, it indicates that the cellview is in read-only mode.

Note: When the cellview is locked by some other user, you will need to wait until the lock is released.
