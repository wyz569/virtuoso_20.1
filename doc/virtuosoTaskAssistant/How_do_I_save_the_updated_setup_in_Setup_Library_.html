Product Version ICADVM20.1, October 2020

How do I save the updated setup in Setup Library?

To save the changes made to the setup elements in the same cellview:

  • Click Save on the toolbar.

To save the changes made to the setup elements in a different Setup Library cellview:

  • Click Save A Copy on the toolbar. 

The Save As Setup Library CellView form appears in which you can specify the library, cell, and view name to save the setup.
