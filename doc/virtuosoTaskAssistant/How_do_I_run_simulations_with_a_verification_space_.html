How do I run simulations with a verification space?

To run the simulation in ADE Verifier while considering the sweeps and corners defined in the verification space, which is time-saving and ensures 100% coverage of the analog design:

  • On the Simulation toolbar, in the Run Mode drop-down list, choose Local with SPACE. If you want to run multiple simulations in batch mode, choose Batch with SPACE.
  • Click .

The simulation for the selected implementation is run using the verification space defined in the Setup Library assistant and the Results tab in ADE Verifier displays the updated results.
