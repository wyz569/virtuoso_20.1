Introduction to Pin Group Guides

Virtuoso® Floorplanner lets you create pin groups and assign pin constraints to these groups. This feature is useful for designs with a large number of pins, such as mixed-signal designs, where it would be difficult and time-consuming to create pin constraints on individual pins. You can create pin groups, assign them to full or partial edge(s), and apply process constraints, such as layer range and spacings, to the entire pin group through a dedicated pin group guide constraint.

A pin group guide constraint:

  • comprises one or more pins and may have one or more pin group guides.
  • is a placement constraint for the pin group, and is defined on partial or complete edges of blocks.

Pins are placed on the pin group guides. In the following example, one full-edge pin group guide and four partial-edge pin group guides are created.

For mixed-signal designs, the pin group guide constraint data is inter-operable with InnovusTM Digital Implementation System (Innovus).

Use the options in the Pin Group Guide form to create and edit pin group guide constraints. These constraints are created in the Virtuoso constraint system. Therefore, you can use the Virtuoso Constraint Manager assistant to verify and further customize these constraints. In the following snapshot, the Constr_2 pin group constraint is selected in the Constraint Manager assistant and its attributes are displayed in the Property Editor. The Constr_2 pin group guide constraint comprises pins B:B and C:C, and a pin guide B__1.

The pin group guide constraint selected in the Constraint Manager assistant is highlighted in the layout design.

After creating the pin group guide constraints, you need to run the Pin Optimizer to achieve the final pin placement. The Pin Planner and Pin Optimizer honor the pin group guide constraints. The Pin Optimizer places the pins accordingly, while maintaining design correctness.

Creating a Pin Group Guide

Virtuoso Floorplanner provides options to create pin groups and guides interactively. When you select pins on the layout, then the pins are selected and highlighted in the form, and vice versa.

For quick reference of this feature, refer to the Pin Group and Guide (PGG) Support in VFP demonstration.

Choose Floorplan – Pin Group Guides – Create to display the Pin Group Guide form.

In Layout EXL, choose Plan – Pin Planning – Create/Edit Pin Group Guide.

The steps for creating a pin group guide are:

  1. Specify the pin group tabulates. Use the following options:
    • Group Name
    • Group Attributes
    • Group Constraints - Global
  2. Select the pins to be included in the pin group guide. Use the options on the PINS tab.
  3. Draw the pin group guide on the layout canvas. Use the options on the GUIDES tab.
  4. Choose the required option:
    • Constraint Only: Updates only the pin group guide constraint.
    • Preview: Provides a visualization of the placement of the pins on the pin group guides.
    • Optimize (default): Updates the pin group guide constraint and places pins on the guide. The pins are optimized to obtain the shortest possible net length.
  5. Choose a Mode. This option is only applicable to the Pin Optimizer.
    The Mode is set to All or Selected depending on whether all or some of the pins are selected.
  6. Click OK or Apply to create the pin group guide.

At any point before creating the pin group guides, you can click Refresh to discard the changes made to the form. Refresh loads the default values from the database, technology file, and constraint. Whenever the database, technology file or constraints are modified, then ensure that you click Refresh to update form values accordingly.

Group Name

To create a new group, either type the new group name in the Group Name field or click New to generate a unique group name. You can edit the automatically-generated group name.

To edit the attributes of an existing group, select the group name from the Group Name drop-down list.

To delete an existing pin group, choose the group name from the Group Name drop-down list and click Delete.

Group Attributes

If you select the Preserve Order check box in the Group Attributes section, then the Pin Optimizer honors the order of pins in the pin group constraint and optimizes the pins in that sequence.

The Foreign Pins Restrictions section in the Group Attributes group box includes the following options:

  • Inclusive: Supports the presence of foreign pins within the pin group guide area. Therefore pin group guides with different pins can overlap in the layout design. The Inclusive option should be selected for overlapping pin group guides.
  • Exclusive: Does not support the presence of foreign pins within the pin group guide area. In this case, pin group guides with either one or both pin group guides set to Exclusive cannot overlap in the layout design. Suitable error messages are displayed in the CIW.

Group Constraints - Global

The Group Constraints - Global section provides the following options to define global constraints for all pins in the pin group:

  • Valid Layers: Specifies valid layers for pin placement. Click Define to display the Layer selection pop-up box, which lists all the layers that are available in the current design.
    Add or remove the layers by selecting or clearing the corresponding check boxes.
    Click to delete all layers from the Valid Layers field.
    If the Layer Priority check box is selected, then pins are optimized based on their layer priority. Layers are prioritized in the order in which they are listed in the Valid Layers field. You can alter the sequence by deleting the layers from the Valid Layers field and typing the layer names in the required sequence, or by selecting layer names in the required sequence from the Layer selection list.
    Either set the matchLabelLayerWithPin environment manually or from the Floorplanning tab of the Floorplan Global Options form (Layout GXL Options form in IC6.1.8) to place pin labels on the same layers as their corresponding pins. Therefore, if a pin is re-layered during placement, then the pin label is also moved to the same layer.

PINS tab

Use the PINS tab to select pin members for the pin group guide constraint.

The PINS tab comprises a pin browser on the left side and a pin group table on the right side. The pin browser enlists all the shape pins found in the current scope. It does not list pins that are vias. The entries are expressed as termName:pinName:figName because the pin figure is the physical entity that you are placing. If a pin has more than two pin figures attached to it, but the pin figures have the same name, they are represented by a single entry in the pin list. Pins that you select in the layout design are automatically selected in the pin browser. Similarly, pins selected in the pin browser are automatically highlighted in the layout design.

Select the required pins in the pin browser and right-click to display the shortcut menu, which provides the following options:

Use to move pins from the pin browser to the pin group table.

After moving pins to the pin group table, right-click to display the shortcut menu in the pin group table, which provides the following options:

You can add top-level pins, soft block pins, or a mix of both to the pin group table. To display a list of soft block pins, choose Soft Block. Select the required soft block from the drop-down list. Pins in the selected soft block are displayed in the pin browser. Also the focus in the layout design changes to the selected soft block.

Pins can be added only once in a pin group guide.

Use the add (>>) or remove buttons to add or remove pins from the pin group table. The pin group table also provides the Pin Name, Width, Spacing, and Status values in corresponding columns. Status indicates the placement status of the pins. Valid statuses are:

  • P - Placed
  • U - Unplaced
  • L - Locked
  • F - fixed
You can directly edit the pin width, spacing, and status values in the pin group table. Double-click the required cell and select the required values from the drop-down list.

Cross-selection of pins is implemented between the layout design and the Pin Group Guide form. Therefore, the pins selected on the PINS tab are highlighted in the layout design. Similarly, the pins selected in the layout design are selected in the pin browser.

GUIDES tab

Use the options on the GUIDES tab to interactively create guides.

If you try to create a pin group guide constraint with no pins, then an error message is displayed in the CIW, and the guide is not created.

You can create the following types of pin group guides:

  • Partial Edge: Use this option to create pin group guides that cover partial edges.
    1. Choose Partial Edge.
    2. Click Create Guide.
    3. In the layout design, click the required edge to indicate the first point of the pin group guide. See Edges for Creating Pin Group Guides for more information.
    4. Click to indicate the last point of the pin group guide. If you are creating multiple pin group guides, continue to click the edges to create the additional pin group guides.
    A pin group guide is not created if the length of the guide is too small for placement of the selected pins, that is if the guide is smaller than the sum of the minWidth, keepout, and spacing values for metal1. A suitable warning message is displayed in the CIW.

    5. Press Esc when you are done.
    6. Choose one of the options Constraints Only, Preview, or Optimize (default).
    7. Choose a Mode.
    8. Click Apply to submit the updates you made and view the pin group guide in the layout design.

Area boundaries are created and the specified pins are placed within the pin group guide that you created.

Width of the pin group guide equals the unit width area boundary.

If you enter a top-level or soft block corner as the first point of a pin group guide, then, by default, you are prompted to create a vertical guide (along the y-axis). To create a horizontal guide, press F3 to display the Create Guide popup box. Choose Horizontal from the Snap Mode drop-down list and click Hide to close the Create Guide popup window. Proceed to create a horizontal pin group guide. You can use this method to create orthogonal pin group guides.

Edges for Creating Pin Group Guides

Choose one of the following edges for creating pin group guides:

  • Partial Edge
  • Full Edge (See Creating a Full Edge Pin Group Guide)
  • Box (See Creating a Box Pin Group Guide)
  • Free Area (Creating a Free Area Pin Group Guide)

Creating a Full Edge Pin Group Guide

Use this option to create pin group guides that cover the edges completely. Unlike partial-edge guides, full-edge guides can be specified with a single click on a top-level or soft block edge in the layout design. The entire edge on which you clicked is selected.

Continue to click the required edges to create multiple full-edge pin group guides.

While selecting the edges, you can go back to the Pin Group Guide form and select Partial Edge to start creating partial-edge guides along with the full-edge guides that you created. The Enter function automatically switches from the Partial Edge to Full Edge mode. Press Esc when you are done. Click Apply to submit the updates you made and to view the pin group guides in the layout design. Pins are automatically placed along the pin group guide that you created.

Use the Offset option to specify the Start and End offset values of the pin group guide on the corresponding edges.

Creating a Box Pin Group Guide

Use this option to draw one or more areas in the layout window. Pin group guides are created on the PR boundary edges that intersect the selected boxes. Press Esc when you are done. If the Split into edge guides during create check box is selected, top-level and soft block edges that intersect the box are created as the partial- or full-edges for creating pin group guides.

If the Split into edge guides during create check box is deselected, then a bounding box is created for the selected area. Pin edges within the bounding box are selected for placing pins.

Whenever the PR boundary of a digital block is modified though stretch or move operations, the area pin group guides on these blocks are automatically snapped to the PR boundary.

Creating a Free Area Pin Group Guide

Use this option to place pins in the free area within the PR boundary of the top level or instance level (level-1) from the top level. To create a free area pin group guide:

  1. Select Free Area on the Guides tab.
  2. Click Create. By default, you can create polygonal guides.
  3. Click F3 to open the Create Guide form.
  4. Select a Free Area Type, which indicates the shape of the guide that you want to create. The following image shows the differences between polygonal and multi-segment guides.
    The top-level (level-0) and level-1 pin group guides are highlighted in different colors on the layout canvas so that they are visually distinguishable.
  5. Select Create Top Design Free Area guide over Soft Block to create top-level (level-0) guides over soft blocks. By default, if soft block pins are added on the Pins tab, then a level-1 guide is created.
  6. Click Hide.
  7. Draw the required shape for the free area pin group guides.
    Free area pin group guides must be completely enclosed within the PR boundary of the block.
  8. Either press Enter or double-click to indicate the end point.
    For polygonal guides, the polygon is auto-completed. Polygons should contain at least four vertices.
    For multi-segment guides, double-click or enter indicates the end point.
    Free area pin group guides must not overlap (partially or completely) with any boundary. If there is an overlap, the pin group guide is not created and a warning message is displayed in the CIW.

A free area pin group guide as per your specifications is created.

Comparing Pin Group Guides

Select Floorplan – Pin Group Guides – Compare to display the Compare Pin Group Guides form.

In Layout EXL, choose Plan – Pin Planning – Compare Pin Group Guides.

Use this form to compare the pin group guides in two different cellviews. Either type the lib:cell:view of the source cellView or click Browse and select the cellView.

Placing Pin Group Guides as per Schematic or Symbol

Virtuoso Floorplanner lets you generate pin group guides and place pins as per their relative positions in the schematic or symbol view. Separate pin group guides are created for each side of the PR boundary, and pins are distributed as per their specified positions in the schematic or symbol view. To create a pin group guide as per the relative positions of pins in the schematic view:

  1. Choose Floorplan – Pin Group Guide – Place As Schematic/Symbol.
    In Layout EXL, choose Plan – Pin Planning – Place Pin Group Guide as Schematic/Symbol.
  2. Choose Schematic or Symbol, as per your requirements. Default is Schematic.
  3. Click OK. The Pin Group Guide form is displayed.
    Group Name lists four different pin group guides, one for each PR boundary edge. Any existing pin group guides and alignment constraints in the design are deleted.
    Pin group guides are created only for the edges that have pins in the schematic view. For example, consider a schematic with pins only at the top and left edges of the PR boundary. Here, only two pin group guides will be created, one at the top edge and the other at the left edge.
  4. Click OK.
