Product Version ICADVM20.1, October 2020

How can I run Tree Route using context-sensitive menu in layout?

To run Tree Route from the layout canvas:

  1. In the layout canvas, select a net.
  2. Right-click the selected net.
  3. Choose one of the routing options from the Net context-sensitive menu.

The selection of the option is dependent on the scope of running Tree Route.
