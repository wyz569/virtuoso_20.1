Tree Route Quick Help
Product Version ICADVM20.1, October 2020

Can I run Tree Route in a specific area?

To run Tree Route in a specific area:

  1. Open a design in the layout window.
  2. In the Navigator assistant, select a net. The instance pins on the selected net are highlighted in the layout window.
  3. In the Wire Assistant, select the design style as Automatic from the Design Style drop-down list.
  4. Select Tree Route as the routing flow from the Route Flow drop-down list.
  5. Ensure that Generate Trunks is selected in the Routing Steps group box.
  6. Select No Mesh from the Generate Trunks drop-down list.
  7. In the Navigator assistant, right-click the selected net.
  8. From the context-sensitive menu, click Select Area and Route with Wire Assistant Overrrides.
  9. With the net still selected in the Navigator assistant, use the mouse to select two points to digitize a box around the area of the net pins.

The routing result will show that only the area that was selected is routed.
