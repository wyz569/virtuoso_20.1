Product Version ICADVM20.1, October 2020

How can I run Tree Route using the Navigator assistant?

To run Tree route from the Navigator assistant:

  1. From the Navigator assistant, select a net.
  2. Right-click the selected net.
  3. Choose one of the routing options from the Navigator context-sensitive menu.
