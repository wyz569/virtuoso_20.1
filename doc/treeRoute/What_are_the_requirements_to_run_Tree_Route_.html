Tree Route Quick Help
Product Version ICADVM20.1, October 2020

What are the requirements to run Tree Route?

The main requirements to run the Tree Route flow are as follows:

  • Virtuoso EXL - Setup to run Layout EXL
  • Pin to Trunk Router – technology file must be set up to run the Pin to Trunk router
  • Ensure that Design Style in the Wire Assistant is selected as Device Level

In case, Tree Route is run on a layout with standard cells, a warning message is displayed stating that the Tree Route flow supports only advanced node device-level designs. This message is also displayed when the design style in Wire Assistant is set to ASIC or Chip Assembly.
