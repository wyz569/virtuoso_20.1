Tree Route Quick Help
Product Version ICADVM20.1, October 2020

How do I define the scope of Tree Route?

You can define the scope of Tree Route.

  • Route all the nets in the design by clicking All in Route Net.
  • Route only the selected nets in the design by clicking Selected in Route Net.
  • Perform area-based tree routing by clicking the Select Area and Route with WA Overrrides option in the Net context-sensitive menu.

In addition, you can also define the type of pins to connect to. 

To select the pin type, click the Pin Type field. You can select the pin type as All, Gate, or Source and Drain.
