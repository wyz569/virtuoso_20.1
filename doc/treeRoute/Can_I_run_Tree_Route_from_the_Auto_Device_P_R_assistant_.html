Tree Route Quick Help
Product Version ICADVM20.1, October 2020

Can I run Tree Route from the Auto Device P&R assistant?

You can enable Tree Route using the Auto Device P&R assistant. To enable Tree Route:

  1. Right-click the empty space in the Virtuoso Layout Suite menu bar.
  2. Choose Assistants – Auto Device P&R.  The Auto Device P&R assistant is displayed in the layout window.
  3. Click Route Devices in the assistant. The Enable Tree Router in Wire Assistant option is displayed. 
  4. Click Enable Tree Router in Wire Assistant.

This enables Tree Route in Wire Assistant, and the Route Flow field in Wire Assistant displays the Tree Route icon.
