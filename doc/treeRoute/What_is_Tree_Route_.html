Product Version ICADVM20.1, October 2020

What is Tree Route?

Tree Route is a new routing flow with more automation and less GUI controls.  This routing flow lets you:

  • automatically route advanced node device-level row-based designs, with a minimum set of options.
  • quickly connect device pins in a structured topology.
  • automatically identify and route mesh, ring (tap cell rings), and tree structures in the design.
