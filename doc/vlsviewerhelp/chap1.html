Product Version ICADVM20.1, October 2020

Overview

This chapter introduces Virtuoso Layout Viewer, an application to open a layout cellview in view-only mode. It includes the following topics:

  • Layout Viewer Features
  • Launching Layout Viewer

Layout Viewer Features

Layout Viewer supports the following features:

  • Layout Viewer is a view-only application. It lets you open a layout cellview in view-only mode. However, you cannot make edits of any type or save a cellview using Layout Viewer.
  • If a layout cellview is opened in edit mode in memory, it cannot be opened using Layout Viewer.
  • If a cellview is opened in Layout Viewer and is then opened in edit mode in Layout XL or Layout EXL, a warning message is displayed in the CIW.
  • Layout Viewer supports the predefined bindkeys for the features available in the application. For more information, see Bindkeys.
  • Layout Viewer supports the SKILL functions for the features available in the application. For more information, see Virtuoso Layout Suite SKILL Reference.
Layout Viewer is not an integration platform, and does not allow any customizations.

Launching Layout Viewer

To launch Layout Viewer, ensure that you have the correct Virtuoso version and license. See Licensing Requirements. The following table describes how you can launch Layout Viewer from Virtuoso.

Location Process to Launch Layout Viewer

Virtuoso Library Manager
  1. Double-click a cellview.
  2. In the Open File form, select the layout view from the View drop-down list.
  3. From the Open with drop-down list, select Layout Viewer.
  4. Click OK.

Virtuoso CIW
  1. Choose File – Open.
  2. In the Open File form, select the library, cell, and view of the type layout.
  3. From the Open with drop-down list, select Layout Viewer.
  4. Click OK.
