License Reason Printing

You can use the -licinfo command-line option with xrun, xmelab, or xmsim commands to determine the simulation-time licenses required for a specified design. The -licinfo option provides additional information about the features that use a DMSO or an Incubation license that is checked out for mixed signal design.

The -licinfo option does not check out any licenses and does not run the simulation. The software only loads the snapshot header to identify the current license requirements.

The DMSO License is needed because some of the following feature sets are used in the design:

The Incubation License is needed because some of the following feature sets are used in the design:

    * Option -relax_svbmuf

The -licinfo option generates a report on the licensing requirements using the following parameters:

  • The contents of the design (AMS, SystemC, SystemVerilog, Verilog, VHDL, and so on).
  • Other options specified on the command line.
  • Any arguments specified using the -uselicense Option.

Debugging features such as Tcl-commands force/release on wreal and save/restart require DMSO; however, this cannot be determined by -licinfo.
