Using Complex Port Definition with User Defined Nettype

You can have complex port connections with System Verilog user-defined nettypes (UDN) in your design by using the -svudn_complex_port option. All the elements in the connections to the ports must be of user-defined nettypes. The direction of the port can be input, output, or inout.

Port expressions can be any of the following:

  • Bit-select of UDN array used in port concatenation. For example:
    module dut ( .aain({aain[1], aain[0]}) );
        input [2:0]  aain;
        cell1 I0 ( .ain(aain));
    endmodule

  • Part-select of UDN array used in port concatenation. For example:
    module dut ( .aain({aain[1:0], aain[3:2]}) );
        input [3:0]  aain;
        cell1 I0 ( .ain(aain));
    endmodule

  • Identifier of UDN used in port concatenation. For example:
    module dut ( .aain({a3, a2}) );
        input a2, a3;
        wire a1;
        cell1 I0 ( .ain({a3, a2, a1}));
    endmodule

  • Combination of above used in port concatenation. For example:
    module dut ( .aain({aain[1], aain[3:2]}) );
        input [3:0]  aain;
        cell1 I0 ( .ain(aain));
    endmodule

  • Bit-select of UDN array used in port definition (no concatenation). For example:
    module dut ( .aain(aain[0]) );
        input [2:0]  aain;
        cell1 I0 ( .ain(aain));
    endmodule

  • Part-select of UDN array used in port definition (no concatenation). For example:
    module dut ( .aain(aain[1:0]) );
        input [2:0]  aain;
        cell1 I0 ( .ain(aain));
    endmodule

The following is an example of a complex port definition:

typedef struct {
  real V;
  real I;
} real_wire_t;

function automatic real_wire_t real_wire_res(input real_wire_t driver[]);
  foreach (driver[i]) begin
    real_wire_res.V += driver[i].V;
    real_wire_res.I += driver[i].I;
  end
endfunction

nettype real_wire_t real_wire with real_wire_res;

module top (  );
    wire [1:0] aain;
    dut I0 ( .aain(aain));
endmodule
module cell1(ain);
    input  real_wire ain[2:0];
    assign ain[0] = '{1, 1.1};
    assign ain[1] = '{2.2, 2.1};
endmodule
