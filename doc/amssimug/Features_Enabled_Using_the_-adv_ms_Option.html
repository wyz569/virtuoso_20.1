Features Enabled Using the -adv_ms Option

You can enable the following functionalities using the xrun/xmelab command-line option, -adv_ms:

  • Connecting SystemVerilog Logic Ports to Electrical Using Wire/Interconnects
  • Connecting SystemVerilog Variable Logic to SPICE Electrical Output Bus

The -adv_ms option is a sub-set of the -sv_ms option; so, all features documented in this section (under -adv_ms) can be enabled using the -sv_ms option, as well. See also, Features Enabled Using the -sv_ms Option.

Connecting SystemVerilog Logic Ports to Electrical Using Wire/Interconnects

You can connect a SystemVerilog (SV) logic port to an Electrical port through a wire or interconnect using the -adv_ms option.

The following types of SV logic port connections are supported:

  • Full bus to VAMS electrical using wire
  • Partial bus/part-select/index part select to VAMS electrical using wire
  • Bit-select to VAMS electrical using wire
  • Concat to VAMS electrical using wire

The wire port connection can be full, partial, bit-select, or concat.

The following example illustrates a SV logic vector connecting to an electrical port through wire:

//top.sv
module top;
 logic [1:0] bus;
 foo f1 (bus);
endmodule

// foo.vams
module foo(e);
output [1:0] e;
 child c0 (e[0]);
 child c1 (e[1]);
endmodule

module child (p);
 output p;
 electrical p;
 real r;
analog begin
V(p) <+ r;
end

initial begin
   r = 0;
   # 10;
   r = 2.2;
end
endmodule

Connecting SystemVerilog Variable Logic Vector to SPICE Bus

You can connect a SystemVerilog logic vector to a SPICE electrical bus by using the -adv_ms option. Then, the tool generates a wire bus port as wrapper and maps the wire bus to the electrical nodes.

The following is an example of an SV logic vector connecting to a SPICE electrical bus:

-------------top.sv-----------

`timescale 1ns/1ps

module top;
   real i[3:0];
   real o[3:0];
    spice_middle x0 ( i,o);
   //initial #3 $display("==== ", i,,o);

Endmodule

------ spice.scs-----------

subckt spice_middle ( i_3  i_2  i_1  i_0 out\<3\> out\<2\> out\<1\> out\<0\>)
v7 ( i_3 0) vsource dc=5
v6 ( i_2 0) vsource dc=5
v5 ( i_1 0) vsource dc=5
v4 ( i_0 0) vsource dc=5
v3 (out\<3\> 0) vsource dc=5
v2 (out\<2\> 0) vsource dc=5
v1 (out\<1\> 0) vsource dc=5
v0 (out\<0\> 0) vsource dc=5
ends

For information about SystemVerilog logic bus to electrical connections, see Connecting SystemVerilog Logic Ports to Electrical Using Wire/Interconnects.
