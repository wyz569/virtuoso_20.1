Enabling Net Aliasing for SV-UDN Connections

The system function $real_net_alias() is used to alias a local net to a hierarchical net through a string reference within connect modules and regular modules.

To enable net aliasing of SV-UDN real net arrays, you must use the -udn_real_net_alias command-line option with xmelab or xrun.

For more information on Node/Net Aliasing System Functions, see the Aliasing Nets to Hierarchical Nets in L2R/R2L Interface Elements section.
