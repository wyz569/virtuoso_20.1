PART 1: Spectre AMS Designer Simulator and Xcelium Simulator with Mixed-Signal Option Capabilities

 This section describes the AMS Designer features that can be run using the Spectre AMS Designer (AMS Designer) simulator and the Xcelium simulator with mixed-signal option. It includes the features related to Verilog-AMS, and VHDL-AMS along with SPICE or Verilog-A, and related use models.

The features covered in Part 1 are common for both the simulators. For more information on the features that can be run only using the Xcelium simulator with mixed-signal option, see "PART 2: Xcelium Simulator Mixed-Signal Capabilities."

This part contains the following chapters:

  • Getting Started with the AMS Simulator
  • Using the Analog Simulation Control File
  • Specifying Controls for the Analog Solvers
  • Using an amsd Block
  • Preparing the Design: Using Analog Primitives and Subcircuits
  • Preparing the Design: Using Mixed Languages
  • Real Number Modeling
  • Using xrun for AMS Simulation
  • Using the AMS Designer Simulator for Design Verification
  • Designing with Multiple Power Supplies
  • Compiling
  • Elaborating
  • Simulating
  • Using SimVision with the AMS Simulator
