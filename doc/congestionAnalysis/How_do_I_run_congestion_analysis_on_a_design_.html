How do I run congestion analysis on a design?

To run global routing and congestion analysis on the design data:

  1. Click the Congestion Analysis icon  on the toolbar.
    You can view the status and the results of the congestion analysis in the CIW or review the data in the log file.

  2. To clear the analysis results and delete global routing, click the Clear Congestion Analysis Data icon on the toolbar.
