What is scenic ratio with respect to congestion analysis?

The scenic ratio determines how close the value is to the ideal route when comparing the route in isolation without any congestion penalties, versus the result when all nets are routed and congestion modifies the taken path. When the value of the scenic ratio is close to 1, it means that the global route is accurate.

The nets on which congestion analysis is not run display the value of the scenic ratio as 0.
