Congestion Analysis Task Assistant

The Congestion Analysis assistant determines the routing capacity of a design and can be run both on globally routed and detail-routed designs. The level of congestion in a design can be represented in the following ways:

  • Graphically using a heat map
  • Statistically using a histogram
  • By using the routing metrics table (used internally by the global routing engine to map capacity and plan routing)
