Congestion Analysis Task Assistant
Product Version ICADVM20.1, October 2020

Is there a workspace available for congestion analysis?

Congestion analysis does have a workspace of its own. The workspace displays the Navigator assistant, Palette assistant, and the Congestion Analysis assistant.

To load the Congestion Analysis workspace, choose Windows – Workspace – Congestion Analysis.
