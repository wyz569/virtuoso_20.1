dbStartGenAnyInstToInstTerm

dbStartGenAnyInstToInstTerm(
d_instId
)
=> I_genId

Description

Returns a dbGenerator object that can be used to iterate over all instance terminals for the specified instance.

Arguments

d_instId

Database ID of an instance.

Value Returned

I_genId

ID of a dbGenerator object.

Example 

gen1 = dbStartGenAnyInstToInstTerm(inst1)
while(instTermId = gen1~>next
;Use instTermId in the code
)
; Close dbGenerator object reference and remove it from virtual memory
dbStopGen(gen1)

dbStopGen

dbStopGen(
I_genId
)
=> t / nil

Description

Forces a dbGenerator object reference to close and removes it from virtual memory.

Arguments

I_genId

Specifies ID of a dbGenerator object.

Value Returned

t

dbGenerator is closed.

nil

dbGenerator is not closed.

Example 

gen1 = dbStartGenNet(cellview1)
while(netId = gen1~>next
;Use netId in the code
)
; Close dbGenerator object reference and remove it from virtual memory
dbStopGen(gen1)
