ICADVM18.1 ISR3

What’s New and Enhanced

Feature

Setting Display Stop Points for maskLayout Databases

New

You can use the maskLayoutStopLimit variable in the hed.env file to specify a size limit for maskLayout cellview databases. If a configuration includes such databases that are larger than the specified size limit, they will not be opened in Hierarchy Editor and will be shown as leaves with display stop point icons.

ICADVM18.1 ISR1

What’s New and Enhanced

Feature

View Bindings for Individual Bits

New

You can now set view bindings separately for individual bits of an iterated instance when using the AMS UNL netlister.
