ICADVM18.1 ISR9

What’s New and Enhanced

SKILL Functions

ipcSleepMilli

New

Causes the parent process to be delayed for the given number of milliseconds.

ICADVM18.1 ISR2

What’s New and Enhanced

SKILL Functions

removeListDuplicates

New

Removes duplicate entries from a SKILL list and returns a new list with the duplicates removed.

sstatus

Enhanced

A new variable has been added: floatPrecisionChars.

ICADVM18.1 ISR1

What’s New and Enhanced

SKILL Functions

getMuffleWarnings

New

Returns a list of warnings that were called and suppressed by the preceding muffleWarnings command.

muffleWarnings

New

Returns the result of the last expression evaluated.

ilgResetWarningMarker

New

Clears a warning marker on a specified line in the SKILL IDE editor window.

ilgSetWarningMarker

New

Sets a warning marker on a specified line in the SKILL IDE editor window.

ICADVM18.1 Base Release

What’s New and Enhanced

SKILL Function

ilgScrollToLocation

New

Scrolls to the specified location in the specified SKILL IDE editor window or tab.
