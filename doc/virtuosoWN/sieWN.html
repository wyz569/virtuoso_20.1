ICADVM18.1 Base Release

What’s New and Enhanced

Spectre Interactive Environment is a new debugging tool introduced to debug Spectre simulations interactively by using events, triggers, and breakpoints. You can run this tool in the standalone mode or from the results of a simulation run in ADE Assembler. For more details, refer to Spectre Interactive Environment User Guide.
