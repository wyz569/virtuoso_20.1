Product Version ICADVM20.1, October 2020

How do I import stimuli?

To import stimuli from other cellviews:

  1. In ADE Explorer, choose Session — Import. 
  2. Select the Library, Cell and View to import their stimuli.
  3. Select the check box, Stimuli.
    The imported stimuli definitions are appended to the existing stimuli definitions in your view.

You can import other information from the design, including the stimuli assignments, while importing stimuli in Explorer. These overwrite the current assignments.
