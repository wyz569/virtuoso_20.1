ADE Stimuli Assignment Quick Help
Product Version ICADVM20.1, October 2020

Is it possible to see waveforms for the same stimulus with different parameter values?

Yes, to do this, when you create a stimulus and generate its waveform previewed in Virtuoso Visualization XL, then follow the steps:

  1.  Make changes in the same stimulus definition.
  2. Select Append in the Wave Status drop-down list.

Click Apply to run the simulation. 
The new waveform is appended in the Virtuoso Visualization XL plot.
