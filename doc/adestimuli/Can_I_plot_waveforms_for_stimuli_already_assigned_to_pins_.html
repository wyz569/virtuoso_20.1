ADE Stimuli Assignment Quick Help
Product Version ICADVM20.1, October 2020

Can I plot waveforms for stimuli already assigned to pins?

Yes, you can plot the waveforms for the stimuli already assigned to pins (including bus bits and global sources), to verify that you have assigned the correct stimuli to the correct pins. Perform the following actions:

  1. Select one or more assigned pins in the Pins Assignment section.
  2. Right-click and choose Plot transient waveform for assigned stimuli.
