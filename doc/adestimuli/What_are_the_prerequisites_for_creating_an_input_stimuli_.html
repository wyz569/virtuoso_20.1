Product Version ICADVM20.1, October 2020

What are the prerequisites for creating an input stimuli?

For an input stimuli, your top-level schematic should contain the following:

  • Input pins for the signals
  • Global name on a signal (such as vdd!) to use power stimuli
  • All sources, for stimulus or for power supply, from the analogLib library
  • refLibs property in the design, if the sources are located in a different library
