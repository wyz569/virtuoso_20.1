Managing Bookmarks

Click Bookmarks in the hamburger menu to add the displayed topic to favorites or to organize your favorites in a specific folder. You can manage all the bookmarked pages by clicking Bookmarks — Manage Bookmarks.


Manage Bookmarks

You can use the Manage Bookmarks option to:

  • Rename a topic
  • Reorder a topic through drag and drop
  • Delete a topic
  • Create new folders or subfolders to sort favorites


The following actions can be performed on the Parent or default node, Bookmarks Bar:

Create

Creates a new node or folder under the default node, Bookmarks Bar, to save the bookmarks under specific nodes. The name of the new node can be set at the same time.

Edit

Edits an existing node. You can move it under a specific parent node with the help of actions Cut and Paste.


Additional actions that can be performed on the child nodes are the following:

Rename

Renames an existing child node or folder.

Delete

Deletes an existing child node or folder.
