Providing Feedback


You can provide documentation related feedback by clicking  in the content viewer pane. It opens the Cadence Help - Submit Feedback form as shown below. 

You can also provide Cadence Help related feedback by clicking the same icon on the Cadence Help Welcome page. 

In addition, you can send usage statistics related to your interactions with the Cadence Help. Select the check box Send usage statistics to Cadence in the Usage Statistics as shown below. This dialog box appears when you launch Cadence Help. Alternatively, you can view this dialog box by clicking Send Usage Stats... in the hamburger menu.
