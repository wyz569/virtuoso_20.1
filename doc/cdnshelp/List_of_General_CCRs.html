Cadence Help User Guide
Product Version 3.12 S004, February 2020

List of General CCRs

CCR 2037982: The Cadence Help window size decreases on clicking the Hide/Show Library button 

Description: The window size of Cadence Help keeps decreasing with every click on the Hide/Show Library button, or the Organize by Document Type and Organize by Products buttons. 

Solution: Readjust the window size manually.

CCR 1269977: Cadence Help should appear in the foreground when a user clicks any Help button

Description: Cadence Help should always open in the foreground when accessed through any Help form or button.

Solution: Look for the Cadence Help instance opened behind the Cadence tool or other open windows, and bring it forward.

CCR 1925037: Cadence Help does not render large HTML pages (73k lines or more)

Description: After QT 5.8. Web Engine porting, the Cadence Help web engine does not render large HTML files containing more than 73k lines.

Solution: Introduce topic-tagging to divide the chapters into separate HTML pages.

CCR 1638319: ‘Go to Top’ icon is missing in the TOC pages if opened from CDNSHelp Doc Server

Description: The new ‘Go to Top’ icon is missing in the TOC pages if docs are opened from the Cadence Help Backend server. This issue occurs only when Cadence Help is accessed through a mapped drive.

Solution: The workaround is to navigate manually, to the top of the page.
