Moving documents from UNIX to Windows


You can move the documents from UNIX to Windows in the following ways:

  • By using the export and import features.
  • By copying the doc directory from Cadence Help UNIX installation to Cadence Help Windows installation and then running the cdnshelp -r command to launch Cadence Help on Windows.
  • By mapping the UNIX drive to Windows. After this you can view the documents using the library feature.
