Cadence Help User Guide
Product Version 3.12 S004, February 2020

Specify the View Settings of the Content

You can use the General tab to set a home page, specify the geometry, font, navigation, and shutdown behavior of the Cadence Help application. 

Home Page

You can specify any HTML page from the doc hierarchy as your home page. This page will be displayed in the Topic Viewer when you click the Home button on the Topic Viewer toolbar. You can specify the home page in the General settings.

Notifications

You can disable the notifications related to unsuccessful search attempts and PDF not found.

Content Display Settings

You can specify the font type, size, font color and background color in this section.

Preview Behavior

You can see the font style and size selected.

You can see more settings and change them according to your preferences by clicking . The following options are displayed:

Geometry

Save the window position

Drag the Cadence Help window to any position and then select Save the Window Position to save the current position of the window. The window will be opened at the saved location in the future.

Save the window size

Change the size of the window and select Save the Window Size to ensure that Cadence Help opens with the specified size in the future.

The view settings cannot be customized for standard pages, such as Welcome Page, History Page, and Results Page.

Shutdown Behavior

Allow Calling Application to Close Cadence Help: Cadence Help will exit when the Cadence tool from which it was invoked is closed.

Library Pane Behavior

Always Expand Documentation

Keep the Documentation Browser in the always expanded mode.

Always Show Library Pane

Keep the Library Pane in the always visible mode.

Weblinks

Open Weblinks in browser: Cadence Help opens the weblinks in an external web browser, outside of the Cadence Help window. This setting also enables the Cadence Online Support links to open in an external web browser. 

Search Preferences

You can set the number of search results to be displayed per page.

If you select Match All Words, the search results with the strings containing the search keyword will be displayed. This is a default setting.

Download File Location

You can change the default download location.

All files (including PDFs) are downloaded at this location if user changes the default download location.
