Specify Web Browser and a Mail Client


You can use the Applications tab to specify a Web browser, specify a mail client, and associated applications to different file types.

Web Browser

You need to specify a Web browser to open HTTP URLs. To specify a Web browser:

  1. Specify the path to the Web browser in the Web Browser box.
  2. Click OK.

Mail Client

Specifies the default mail client. You can browse and select the mail client application that you want to see as the default.
