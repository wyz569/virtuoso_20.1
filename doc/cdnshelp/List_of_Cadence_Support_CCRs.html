List of Cadence Support CCRs

CCR 2131465: Login button click on a Cadence Support Lite page should open the COS login page before showing the content  

Description: When you click the login button on the top right corner of a Cadence Support Lite search result page, the same page opens up in the default web browser. 

Solution: Click the Login button on the web browser page again to open up the Cadence Support login page.

CCR 2118592: Cadence Help should be able to show PDF documents on Cadence Support 

Description: If the search result in Cadence Help opens a Cadence Support link containing a PDF file, it shows up a blank window. This is because Cadence Support is unable to render PDF files.

Solution: You can choose to open such Support Portal pages from Cadence Help search results in an external web browser. To set the browser,  go to Settingsâ†’ General and select Open Weblinks in browser. It sets your default web browser to open a page containing PDF files. After this setting, click the search result again.
