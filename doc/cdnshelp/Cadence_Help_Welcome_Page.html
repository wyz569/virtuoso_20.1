Cadence Help Welcome Page


Cadence Help Welcome page is displayed whenever Cadence Help is launched in the standalone mode. It includes links to the content and videos of the latest and core features. It enables you to know about the latest updates in the tool. This page can also be accessed by clicking the home () icon on the Topic Viewer toolbar. The page also includes an alert carousel. It displays the community content on the different tabs.
