Running Density Analysis

This chapter includes the process of running Density Analysis from Virtuoso and generating Histograms and Density heatmaps.

The content of this chapter is organized as follows:

  • Introduction to Pegasus Interactive Density Analysis
  • Running Density Analysis
  • Running Density Analysis on Visible Layers
  • Merging Density Analysis Results

Introduction to Pegasus Interactive Density Analysis

Density analysis generates density results from the Pegasus Interactive toolbar for the required layers. Rule deck is not required. Results are shown in the layout viewer as density heatmap and in Pegasus Results Viewer as histogram.

You can customize the results per your requirements using the options available in the Density Analysis tab of the Pegasus Interactive Run Options form:

Figure 7-1 Density Analysis Tab

For detailed description of the fields of the Density Analysis tab, see Run Settings for Density Analysis.

Running Density Analysis

Perform the following steps to do density analysis:

  1. Click the Pegasus Interactive Run Options icon on the Pegasus Interactive toolbar. The Pegasus Interactive Run Options form opens.
  2. Select the Density Analysis tab.
  3. Make the required selections in the Density Analysis form as per your requirement, such as for which layers you want to run density analysis, specifications for the histogram to be generated, and density setup.
  4. Click OK to close the Pegasus Interactive Run Options form.
  5. Select Density Analysis from the Pegasus Interactive toolbar.
  6. Click the Run Pegasus Interactive Verify Design icon. On completion of the run, the density heatmap is shown on the layout canvas:
    The Pegasus Results Viewer also opens showing the generated histogram:

In the above steps, density analysis was run on the active layer. Similarly, you can run density analysis on the visible and custom-defined layers.

Running Density Analysis on Visible Layers

Perform the following steps to run density analysis for the visible layers:

  1. Set the visibility of the layers for which you want to run density analysis. In this example, two layers metal1_drawing and metal1_fill are set to visible.
  2. Click the Pegasus Interactive Run Options icon on the Pegasus Interactive toolbar. The Pegasus Interactive Run Options form opens.
  3. Select the Density Analysis tab.
  4. Select the Visible Layers radio button in the Layer Visibility section.
  5. Click OK to close the Pegasus Interactive Run Options form.
  6. Select Density Analysis from the Pegasus Interactive toolbar.
  7. Click the Run Pegasus Interactive Verify Design icon. On completion of the run, the density heatmap is shown on the layout canvas:
    The Pegasus Results Viewer also opens showing the generated histograms. The Pegasus Results Viewer shows two tabs for each layer showing two histograms:
    Figure 7-2 Histogram 1 for Metal1_drawing layer
    Figure 7-3 Histogram 2 Metal1_fill layer

Merging Density Analysis Results

You can merge the run results of multiple layers using the Merge option in the Density Analysis tab. In this case, the tabs are merged into one with Pegasus Results Viewer and histogram show result under one tab only.

  1. Set the visibility of the layers for which you want to run density analysis. In this example, two layers metal1_drawing and metal1_fill are set to visible.
  2. Click the Pegasus Interactive Run Options icon on the Pegasus Interactive toolbar. The Pegasus Interactive Run Options form opens.
  3. Select the Density Analysis tab.
  4. Select the Visible Layers radio button in the Layer Visibility section and also select the Merge check box.
  5. Click OK to close the Pegasus Interactive Run Options form.
  6. Select Density Analysis from the Pegasus Interactive toolbar.
  7. Click the Run Pegasus Interactive Verify Design icon. On completion of the run, the density heatmap is shown on the layout canvas:
    The Pegasus Results Viewer opens showing the generated merged histogram:
