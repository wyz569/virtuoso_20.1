SDR Toolbar

SDR is an effective way to find and display nets with accurate current and resistance estimation.

The SDR toolbar is designed to help you with the various simulation driven interactive routing features and settings. The SDR toolbar is installed automatically when you launch Layout EXL from the layout or schematic view. In case it is not available by default in Layout EXL, you can open it using Windows – Toolbars – SDR Toolbar. The icons on the SDR Toolbar let you access SDR features with a single click.

Environment Variable: weSdrSourceSinkMap

  • Hide Sources and Sinks Map
    Clicking the arrow next to the button displays three options, as explained in the following table.
    Icon Command Lets you

    Display Sources and Sinks Map Clusters

    Shows the current distribution across pin clusters as a thermal map.

    Display Sources and Sinks Map Pins

    Shows the current distribution on a single pin as a thermal map.

    Hide Sources and Sinks Map

    Hides the thermal map showing current distribution across pins and clusters.
    The tooltip on the button is updated based on the selected option. By default, the Hide Sources and Sinks Map option is selected.
  • Interactive SDR Mode
    Specifies if the edited wire width should be calculated according to the current or the maximum resistance to avoid EM or maxResistance violations.
    Icon Command Lets you

    Interactive SDR Current Density Check

    Estimates the current in the edited wire and calculates the wire width and via cuts to avoid EM violations.

    Interactive SDR Max Resistance Check

    Estimates the resistance in the edited wire and calculates the wire width and via cuts to avoid maxResistance violations.
  • EM Width Multiplier
    Helps in increasing or reducing the wires width compared to the estimated width required to satisfy the EM value.
  • Checker Mode
    Lets you choose the SDR checker mode to be used for current and resistance estimation.
    Icon Command Lets you

    Enforce

    Estimates the current or resistance in the edited wire and vias according to the EAD settings (dataset, temperature, and current scaling) and automatically calculates the wire width and via cuts to avoid EM or maxResistance violations.

    Notify

    Estimates the current in the edited wire using a color code based on EAD EM violations color settings. The color is computed to represent the EM or maxResistance violation that is reported by the EAD checker after routing.

    Off

    Retains the existing wire width and there is no feedback on estimated current or resistance.
  • Current Estimation Mode
    Controls how the current in the edited wire and via is estimated and calculated.
    Icon Command Lets you

    Auto

    Estimates the current automatically in the last section of the wire according to the connected objects and flightline targets.

    Sum Connected Pins Current

    Sums up the current of all the physically connected pins.

    Maintain Constant Current

    Inherits the current from the previous wires.

    Nearest Island Current

    Uses the current of all the pins connected by the nearest flightline.
  • Automatically Connect Twigs
    Toggles the twig modes between ON and OFF.
  • Import DI as Dataset
    Imports or updates the EAD dataset called DI with the design intent information.
  • Options...
    Displays the EAD options form.
