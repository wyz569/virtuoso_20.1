    This video shows how to use Virtuoso Symbolic Placement of Devices to quickly edit selected PMOS and NMOS devices in a design, preview the updates made, and then generate a layout.

    This video shows how using multiple row patterns in SPD helps in optimizing a design.
  • Creating Signal Trunks in SPD for Pin-to-Trunk Routing
    This video shows how to create signal trunks in SPD and then use them for Pin-to-Trunk routing in Layout XL.
  • Using User-Defined Abutment and Callback Functions in SPD
    This video shows how user-defined abutment and callback functions can be used in SPD.
