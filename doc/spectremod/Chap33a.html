Model Concepts

The definition of the node potentials are:

Vs: reference

Vgs: gate to source

Vds: drain to source

Ves: substrate contact to source

Vbcs: body contact to source

The internal node potential is denoted by Vbs, which is equal to for the floating body and to Vbcs

for the body tie.

The divergence of model branches are summarized are shown in the figure below

Figure 32-1 Model Branches Considered in HiSIM_SOI

The dynamical-depletion condition during switching is also automatically done within the model framework. The impurity concentration difference between the SOI layer NSUBS and the substrate NSUBB determines the built-in potential difference, which is also considered.

Figure 32-2 Definitions of device parameters in HiSIM SOI.

The y-axis is defined along the direction of current flow (source is y=0) and x is defined downwards into the channel (front gate MOS interface is x=0).

If TBOX is so thin that the inversion condition occurs even on the backside at BOX, HiSIM SOTB is recommended to apply. HiSIM SOI solves the Poisson equation iteratively from the surface to the bulk substrate [1–3]. The surface potential value, the solution of the Poisson equation, determines the device condition either fully depleted (FD) or partially depleted (PD).

Model Usage

Determination of the node sequence is:

Drain = 1, Gate = 2, Source = 3, Substrate = 4

If COBCNODE = 1 Body = 5, Thermal = 6

If COBCNODE = 0 Thermal = 5

The effective channel length LLeff and width Weff are calculated from the gate length Lgate and width Wgate, where Lgate, and Wgate are related to the drawn gate length Ldrawn and width Wdrawn as:

Lgate = Ldrawn

Wgate = Wdrawn/NF

Leff = Lgate - 2 × XLD

Weff = Wgate - 2 × XWD

Weffc = Wgate - 2 ·XWDC

where XLD and XWD account for the overlaps of source/drain contact and the gate oxide. The model parameter XWDCis introduced to describe the different width dependence of capacitances from currents. If the value is not given, the same value as XWD is taken. NF is the number of fingers.

Version Update and Enhancement

Version 1.40 Enhancement

The 1.40 version of HiSIM_SOI contains the following changes:

Enhancement

  • Channel length offset parameter for capacitance (XLDC) is introduced.
  • Pocket impurity concentration NSUBP will be forcefully set to NSUBS when NSUBP is less than NSUBS. This takes into effect when VERSION > 1.30. (Default: 1.40).
  • Pb2 refers to Nsub, SOI impurity concentration with various effects included, such as pocket implantation and STI-induced variation, when VERSION > 1.30 (default: 1.40). When VERSION is less than or equal to 1.30, it refers to NSUBS.
  • L-scaling function for SOI impurity concentration is introduced. New model parameters: NSUBCL, NSUBCLP.
  • SOI resistance model is introduced. New model flag: CORBULK. New model parameters: RBULK0, RBULKW. New OP variable: Rbulk.
  • Deep punchthrough current model is introduced. New model flags: COPT, COPSPT. New model parameters: XJPT, NJUNC, MUPT, VFBPT, PSLIMPT. To activate this model, set COPT=1 and MUPT > 0.0. Previous punchthrough current model continues to exist in the new name “shallow (surface) punchthrough”. To activate this, set COPT=1 and PTL > 0.0.

Version 1.30 Enhancement

The 1.30 version of HiSIM_HV contains the following changes:

  • Changed Model Descriptions:
    • The source codes of floating-body and body-tie were unified by using $port_connected. A new flag is introduced for model selection
      COBCNODE = 0: Floating Body model (default)
      = 1: Body Tie model
  • Code Improvement: ddx() functions were removed from prev calculation
