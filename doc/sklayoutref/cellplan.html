Cell Planner Functions (IC6.1.8 Only)

This section provides syntax, descriptions, and examples for the Cadence® SKILL functions associated with the cell planner tool.

Only the functions documented in this chapter are supported for public use. All other functions, regardless of their name or prefix, and undocumented aspects of the functions described below, are private and are subject to change at any time.

List of Cell Planner SKILL Functions

  • nclCPSaveConstraint
  • nclCPToolBarSelect
  • nclCPExit

nclCPSaveConstraint

nclCPSaveConstraint(
[ d_cellViewID ]
)
=> t / nil

Description

Saves a cell planner Layout Structure constraint on the cellview the cell planner was started from.

Arguments

d_cellViewID

Database ID of the cellview.

Value Returned

t

The Layout Structure constraint was saved.

nil

The Layout Structure constraint was not saved.

Example

Saves onto the source cellview the cell planner Layout Structure constraint.

nclCPSaveConstraint()

nclCPToolBarSelect

nclCPToolBarSelect(
[ s_mode ]
)
=> t / nil

Description

Selects the action to perform in Cell Planning mode (one of: horizontal, vertical, merge, move). If you select horizontal, subsequent clicks create horizontal partitions in the Cell Plan. If you select vertical, subsequent clicks create vertical partitions. If you select merge, subsequent clicks on existing partitions removes those partitions. If you select move, you can move instances between rooms.

Arguments

s_mode

The Cell Plan mode.

Valid values are horizontal, vertical, merge, move.

Value Returned

t

The action was performed on the cell plan.

nil

The action was not performed on the cell plan.

Example

Puts the Cell Planner into the mode to create vertical partitions.

nclCPToolBarSelect("vertical")

nclCPExit

nclCPExit(
)
=> t / nil

Description

Exits Cell Planning mode and returns the toolbars and assistants to their original state.

Arguments

None

Value Returned

t

Cell Planning mode was quit and the toolbars and assistants were returned to their original state.

nil

Cell Planning mode was not quit. Toolbars and assistants remain in Cell Planner mode.

Example

nclCPExit()
