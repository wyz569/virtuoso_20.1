Overview

Cadence® Virtuoso® Power System (VPS) products are sold as part of a product package. Each product has a corresponding license.

The software uses licenses to determine the available features when the software runs.

Checking Out Licenses for VPS Products

The VPS products are integrated in the Virtuoso® Design Environment. These products use a form-based approach to drive the analysis based on user requirements. The results of the analysis are displayed in the Virtuoso GUI, overlaid on the design layout.

Running VPS products requires check-out availability of the Cadence® Design Framework II license (known as the "product 111 license"). Once a DFII license check is performed successfully, you will then be able to access VPS functionality.

For information on licenses for the Virtuoso® Design Environment, see Virtuoso Design Environment Licensing Setup in the Virtuoso Software Licensing and Configuration Guide.

The VPS products leverage existing Cadence tools for performing extraction and simulation but do not include licenses for these tools. These licenses must be purchased separately.

For extraction, you require a license for the following:

  • Quantus QRC version 9.1.2 and above

For simulation, you require a license for one of the following:

  • Spectre version 10.1 and above
  • UltraSim version 10.1 and above
  • APS version 10.1 and above
  • ViVA version IC 6.1.5 and above
  • ADE L version IC 6.1.5 and above
