How do I check for color violations?

To check for color violations:

  1. Click the Checks icon or select Violation Checks from the Checks option.
  2. In the Color Checks form, in the Type section select the types of color checks that you need to perform.
  3. Select the Scope: Current Editable Cellview, Current Visible Area, or Area. 
  4. In the Layers section, select specific layers or the All check box.

The Verify Process Rules Summary is displayed in the CIW.

The table below lists the coloring checks applicable to the three MPT flows.

MPT FlowColor Checks Applicable
Fully Colored Color Shorts, Uncolored Shapes, Unlocked Shapes, and Colorability
Partially Colored Color Shorts and Colorability
Lockless Color Shorts, Uncolored Shapes, and Colorability
