How do I set up a fully colored flow?

You can use the following SKILL functions to set up, check, and report the settings of a fully colored MPT flow:

  • mptSetFlow: To set up a fully colored MPT flow. Use the function as follows: 
    mptSetFlow("FullyColoredAndLocked")
  • mptCheckFlow: To check if the fully colored flow has been set up. Use the function as follows: 
    mptCheckFlow("FullyColoredAndLocked")
    INFO: Flow 'FullyColoredAndLocked' is properly set.
    t
  • mptReportCurrentSettings: To display a report of the MPT flow settings. Use the function as follows: 
    mptReportCurrentSettings(geGetEditCellView()) 
    This function displays the settings of the MPT flow in a *.rpt file.
