Power IR/EM Known Problems and Solutions
Product Version ICADVM20.1, October 2020

About This Manual

This Known Problems and Solutions manual describes the known issues with the Virtuoso Power System (VPS) tool, Power IR/EM, in the Virtuoso Platform Suite and suggests the workarounds for these issues. Each issue is identified as a Cadence Change Request (CCR) number.

Only the problems and solutions that were known at the time of release are described in this document. Log on to Cadence Online Support to view an up-to-date list.

For information about the customer issues that were fixed in theICADV12.1release, do one of the following:

  • Open the README and README.OA files from your installation hierarchy.
  • Open the README and README.OA files by following these steps:
    1. Log on to Cadence Online Support and from the home page, choose Software Updates – Download Software. This opens the Cadence Downloads page. Alternatively, you can visit downloads.cadence.com.
    2. From the list of releases, click the required release name.
    3. From the Base Release, Updates, or Hotfix section, as applicable, click the README and README.OA file links.
