Introduction to Virtuoso Variation Option

Overview

Virtuoso Variation Option is a special license required to run a set of advanced statistical capabilities from Virtuoso ADE Explorer or Virtuoso ADE Assembler environment. If available, the required license, 95265, Virtuoso Variation Option, is automatically checked out when you run simulations using the following run modes:

    • Advanced options of the Monte Carlo run mode
    • Improve Yield
