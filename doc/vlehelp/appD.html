via definition

Cyclic field. Choices are:

== and !=

Cyclic field. Choices are populated from the technology file.

If you select others from the cyclic field, the text field next to it gets enabled.

You can then enter a via name and combine it with a wild card character to get multiple matches. For example, if you enter M2_M1* as the search criterion, all vias with M2_M1 in their names are displayed.
