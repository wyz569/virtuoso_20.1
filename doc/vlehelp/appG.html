Toolbar Icons

  • File Toolbar:
    The following icons are available on the File toolbar:
    • Open
    • Save (Unavailable in read-only mode)
  • Edit toolbar:
    The following icons are available on the Edit toolbar:
    • Undo (Unavailable in read-only mode)
    • Redo (Unavailable in read-only mode)
    • Move (Unavailable in read-only mode)
    • Copy (Unavailable in read-only mode)
    • Stretch (Unavailable in read-only mode)
    • Delete (Unavailable in read-only mode)
    • RepeatCopy (Unavailable in read-only mode)
    • Edit Properties
    • Align
    • Rotate
    • Rotate Left (Unavailable in read-only mode)
    • Rotate Right (Unavailable in read-only mode)
    • Flip Horizontal (Unavailable in read-only mode)
    • Flip Vertical (Unavailable in read-only mode)
    • Create Group (Unavailable in read-only mode)
    • Ungroup (Unavailable in read-only mode)
  • Align Toolbar
    The following icons are available on the Align toolbar:
    • Align Left
    • Align Vertical
    • Align Right
    • Align Top
    • Align Horizontal
    • Align Bottom
    • Align Spacing
    • Orthogonal Spacing
    • Use Reference Layer
  • View Toolbar
    The following icons are available on the View toolbar:
    • Zoom In
    • Zoom Out
    • Zoom to Fit
    • Zoom to Selected
  • Create Toolbar
    The following icons are available on the Create toolbar:
    • Create Instance
    • Create Label
    • Create Pin
    • Create Via
    • Create Wire
    • Create Bus
    • Zoom to Selected
  • Optimize Toolbar
    The following icons are available on the Optimize toolbar:
    • Execute Flow
    • Preferences
    • Browse Constraints
    • Copy Back to Layout
  • Search Toolbar
    The following icons are available on the Search toolbar:
    • Search
  • Options Toolbar
    The following icons are available on the Options toolbar:
    • Full Select
    • Path Spine
    • Via Stack
    • Create/Edit Snap Mode: Orthogonal
    • Transparent Group
    • DRD Enforce
    • DRD Notify
    • DRD Post-Edit
    • Constraint Aware Editing (Unavailable in read-only mode)
    • Update connectivity information when design is modified (Unavailable in read-only mode)
    • Selection Granularity (Unavailable in read-only mode)
    • Info Balloon
    • Display Stop Level
    • Create Snap Mode: Orthogonal
    • Edit Snap Mode: Orthogonal
    • Dimming
    • Snap Wires to Manufacturing Grid
    • Dynamic Measurement
    • Highlight Aligned Edges
    • Layer Scope
  • Status Toolbar
    The following icons are available on the Status toolbar:
    • Selection Mode
    • Number of nets selected (Sel(N):n)
    • Number of instances selected (Sel(I):n)
    • Number of objects selected (Sel(O):n)
    • X and Y
    • dX and dY
    • Distance
  • Guardring Toolbar
    The following icons are available on the Guardring toolbar:
    • Create Guardring by Wrapping
    • Create Guardring from Path
    • Create Guardring from Rectangle
    • Create Polygon Contact Fill
    • Convert to Polygon
    • Chop
    • Merge
    • Tunnel
    • Heal
    • Reshape
    • Split
    • Clean Overlapping Contacts
    • Clean Guardring Cache
  • Show Selection Info Toolbar
    The following icons are unavailable on the Show Selection Info toolbar:
    • PreSel/Select
    • Object Name
    • Attributes
  • Go Toolbar
    The following icons are unavailable on the Go toolbar:
    • Back (Unavailable in read-only mode)
    • Forward (Unavailable in read-only mode)
    • Up (Unavailable in read-only mode)
    • Top (Unavailable in read-only mode)
  • Workspaces Toolbar
    The following icons are available on the Workspaces toolbar:
    • Workspace Configuration
    • Save Workspace
    • Toggle Assistants Visibility
