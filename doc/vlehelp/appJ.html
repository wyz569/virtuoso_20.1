Layout Editor Plug-Ins

This topic lists the plug-ins available in the Virtuoso® Layout Suite cockpit. Plug-ins are applications that can be dynamically added to the layout editor user interface, used to perform some specialized design task, and then removed again when no longer required.

The Launch - Plugins menu contains all the applications that are available from the layout editor. To use any application listed in the menu, you must first load it.

When you load a plug-in application, it appears in the layout editor, typically as a separate menu on the menu bar, but sometimes as a user interface docked on the left-hand side of the layout window. Some plug-ins open the related applications directly on the layout canvas itself.

The figure below shows the Plugins submenu available in the Launch menu of Virtuoso Layout Suite.

The following table lists the plug-ins available in layout editor and provides links to further information on the association feature.

Plug-In

Description

Debug Abutment

Lets you debug the abutment using the Pcell IDE window.
For more information, see Debugging Pcells Abutment in the Virtuoso Parameterized Cell Reference.

Debug CDF

Lets you debug CDF callback procedures using simple SKILL expressions for form initialization (forminit), form completion (doneProc), and any parameter field value modification (parameter callbacks).
For more information, see Debugging CDF in the Virtuoso Parameterized Cell Reference.

Innovus

Adds the Innovus menu to the layout editor menu bar. The Innovus Implementation System is used as the design cockpit if a Verilog netlist is used for the top-level design.
For more information, see the Mixed Signal (MS) Interoperability Guide.

Pcell

Adds the Pcell menu to the layout editor menu bar to let you create graphical Pcells.
For more information, see Creating Graphical Parameterized Cells in the Virtuoso Parameterized Cell Reference.

Simulation: NC-Verilog

Opens the Virtuoso Verilog Environment for NC-Verilog Integration application.
For more information, see the Virtuoso NC-Verilog Environment User Guide.

Simulation: Other

Adds the Simulation menu to the layout editor menu bar.
For more information, see the Virtuoso NC-Verilog Environment User Guide.

Simulation: Spectre

Adds the Spectre menu to the layout editor menu bar. This application provides a solution for digital circuit designers to netlist a Virtuoso transistor-level schematic for the Spectre circuit simulator.
For more information, see the Virtuoso Schematic Editor Spectre and Ultrasim Interface User Guide.

Simulation: UltraSim

Adds the UltraSim menu to the layout editor menu bar. This product provides a solution for digital circuit designers to netlist a Virtuoso transistor-level schematic for the Virtuoso UltraSim circuit simulator.
For more information, see the Virtuoso Schematic Editor Spectre and Ultrasim Interface User Guide.

VPS-L

Adds the VPS-L menu to the layout editor. The Virtuoso Power System L application provides a view of IR drop, electromigration (EM), and signal net EM analysis.
For more information, see the Power IR/EM User Guide.

Voltus-Fi-L

Opens the Voltus-Fi-L application, which uses the extracted view to perform IR drop and Electromigration (EM) analysis.
For more information, see the Voltus-Fi Custom Power Integrity Solution User Guide.

Voltus-Fi-XL

Opens the Voltus-Fi-XL application, which performs multi-mode simulation (MMSIM), electromigration, and voltage drop (EMIR) analysis for power and signal nets using the simulation database generated by Spectre APS/XPS.
For more information, see the Voltus-Fi Custom Power Integrity Solution User Guide.
